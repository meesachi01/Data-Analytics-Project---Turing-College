# 🗄️ Relational Data Exploration (SQL)

This project was completed as part of the **Turing College Data Analytics Program** using **Google BigQuery**.

The objective was to explore a relational database by writing SQL queries to answer business questions, validate query results, and gain insights from interconnected datasets.

## Project Scope

The project focused on:

- Product overview and analysis
- Reviewing work orders
- Query validation and data quality checks

Throughout the project, I applied relational database concepts, including joins, filtering, aggregation, grouping, and query validation to ensure accurate and reliable results.

## Skills Demonstrated

- SQL (Google BigQuery)
- Relational Database Exploration
- INNER, LEFT & RIGHT JOINs
- Aggregate Functions
- GROUP BY & HAVING
- CASE Statements
- Query Validation
- Business Data Analysis
