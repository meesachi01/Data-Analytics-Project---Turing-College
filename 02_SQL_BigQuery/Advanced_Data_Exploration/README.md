# 🗄️ Advanced Data Exploration (SQL)

This project was completed as part of the **Turing College Data Analytics Program** using **Google BigQuery**.

The objective was to solve more advanced business problems by writing complex SQL queries and transforming relational data into meaningful business insights.

## Project Scope

The project focused on:

- Customer overview and segmentation
- Sales reporting and performance analysis

The analysis involved combining multiple tables, creating business metrics, and producing reports using advanced SQL techniques.

## Skills Demonstrated

- SQL (Google BigQuery)
- Common Table Expressions (CTEs)
- Window Functions
- Complex JOINs
- Aggregate Functions
- CASE Statements
- Date & Time Functions
- Query Optimization
- Business Reporting
