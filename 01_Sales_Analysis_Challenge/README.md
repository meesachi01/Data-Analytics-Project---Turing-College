# 📊 AdventureWorks Sales Analysis

This project was completed as the first graded assignment during the **Turing College Data Analytics Program** using **Google Sheets**.

The objective was to analyze AdventureWorks sales data and create a dashboard that provides business insights, including:

- Monthly sales trends
- Online vs. offline sales comparison
- Delivery performance
- Top salespeople
- Top countries
- Top sales reasons

The analysis followed the complete data analytics workflow: **Ask → Prepare → Process → Analyze → Share → Act**.

## Tools
- Google Sheets
- Pivot Tables
- Charts & Dashboards
