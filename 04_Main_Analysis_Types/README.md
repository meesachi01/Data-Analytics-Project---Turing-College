# 📈 Main Analysis Types

This folder contains projects completed during the **Turing College Data Analytics Program** that focus on key analytical techniques used to understand customer behavior, evaluate business performance, and support data-driven decision-making.

## Projects

### 1. Cohort Analysis
Analyze customer retention and purchasing behavior over time by grouping customers into cohorts based on shared characteristics or acquisition periods.

### 2. Funnel Analysis
Measure customer progression through key stages of the user journey to identify conversion rates and opportunities to optimize the customer experience.

### 3. Customer Segmentation & RFM Analysis
Segment customers using **Recency, Frequency, and Monetary (RFM)** analysis to identify high-value customer groups and support targeted marketing strategies.

### 4. A/B Testing Analysis
Evaluate experimental results using statistical methods to compare different variants and determine whether observed differences are statistically significant.
