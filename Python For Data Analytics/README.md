# 🐍 Python for Data Analytics

## Overview

This folder contains Python projects focused on data exploration, visualization, and predictive analytics. Using libraries such as **Pandas**, **NumPy**, **Matplotlib**, and **Scikit-learn**, these projects demonstrate the complete data analysis workflow—from data cleaning and exploratory analysis to machine learning and model evaluation.

## Projects

### 1. Spotify Data Analysis

Exploratory Data Analysis (EDA) of Spotify music data to uncover trends in audio features, artist popularity, genres, and listener preferences through data visualization and statistical analysis.

### 2. Coursera Data Exploration

Performed data cleaning, transformation, and exploratory analysis on Coursera course data to identify trends in course ratings, enrollments, difficulty levels, and subject popularity.

### 3. Cardiovascular Disease Prediction

Developed and evaluated machine learning models to predict the likelihood of cardiovascular disease based on patient health indicators. The project includes data preprocessing, feature engineering, model training, and performance evaluation.
