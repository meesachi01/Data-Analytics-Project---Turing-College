# 📚 Coursera Dataset Analysis

## Overview

This project explores a Coursera course dataset using Python to identify patterns in course ratings, enrollments, difficulty levels, providers, and certificate types. Through exploratory data analysis (EDA) and statistical techniques, the project uncovers the factors that influence course popularity and learner engagement.

The project includes both a detailed Jupyter Notebook and a summarized report presenting key findings and visualizations.

## Business Value

Understanding the characteristics of successful online courses helps educational platforms and content providers to:

- Identify factors that drive course popularity
- Understand learner preferences and engagement
- Optimize course offerings and pricing strategies
- Improve content recommendations
- Support data-driven decision-making for course development

## Key Insights

- Beginner-level courses attract the largest number of learners.
- Course ratings are heavily concentrated between **4.5 and 5.0**, indicating consistently high user satisfaction.
- Enrollment distribution is highly skewed, with a small number of courses accounting for the majority of enrollments.
- More advanced courses generally receive fewer enrollments.
- Established course providers tend to achieve greater popularity.
- Career-focused subjects consistently attract the highest learner engagement.
